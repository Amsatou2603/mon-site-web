Questions de TP
1. Différence entre Git et GitHub
Git est un logiciel de contrôle de version installé localement sur ton ordinateur.
GitHub est une plateforme en ligne qui héberge des dépôts Git sur internet. 

2. À quoi sert git init ?
Elle initialise un nouveau dépôt Git dans le dossier courant. Elle crée un dossier caché .git/ qui contient tout l'historique et la configuration du projet.

3. Quelle commande envoie un projet sur GitHub ?
git push origin main

4. Pourquoi utilise-t-on les branches ?
Les branches permettent de travailler en parallèle sans casser le code principal. 

On fusionne (merge) la branche quand le travail est terminé et validé.

5. Voir l'historique des commits
git log
